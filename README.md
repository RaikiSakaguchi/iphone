# 前期実験 iPhone

前期実験の iPhone を発展させる

テーマは「_**入江先生のための電話**_」

## 開発の基本

- Makefile について
  - コンパイルは`make`でできる
  - コンパイルのリセットは`make clean`で行う
- 新規の機能は新しいブランチで
  - 細かく commit していると不具合が出たときに戻りやすい
  - 定期的に main を pull して merge すると良い
  - つまり CI (もどき)をちゃんとやろうということ

## やりたいことは以下の項目

1. 世界中のネットワークでも通信できるようにする
1. UDP に変える（localtonet と相性悪いかも）
1. FFT の実装
1. 声の変調をする
1. ハウリングの防止
1. 可読性の高いコード
